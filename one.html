What is HTML?

HTML stands for HyperText Markup Language. It is used to structure web pages. HTML elements tell the browser how to display content.

Structure of HTML:

Here’s the basic structure 👇

<!DOCTYPE html>
<!html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>

Explanation of Each Part:

1.<!DOCTYPE html>:
  • Tells the browser that the document is written in HTML 5.
  • Must be the fifst line in an HTML file.
2.</html> tag:
  • The root element of the HTML page.
  • All content of the web page goes inside this tag.
3.  <head> section:
  • Contains meta-information about the web page (not shown on the page itself).
  • Includes:
  • <title> → Defines the page title shown on the browser tab.

    • Example:

    <head>
        <title>Let's Learn</title>
    </head>
    <body>
        <h1>See the website name.</h1>
    </body>

    Out put:

  • <link> → Connects external stylesheets.
  • <script> → Adds JavaScript.
4.<body> Section:
  • Contains all the visible content shown on the webpage, such as:
  • Text (<p>,<h1>-<h6>).
  • Images <img>
  • Links <a>
  • Lists, tables, forms, etc.
5.</html>
  • Closes the HTML document.

HTML Tags:

  • Definition:Tags are the building blocks of HTML. They tell the browser how to display the content.

Syntax:

<tag name> Content </tag name>

Example:

<p> This is paragraph </p>
  • <p> → Opening tag
  • </p> → Closing tag
  • This is paragraph → Content

Types of tags

a.Paired (Container) Tags: Have both opening and closing tags.
Example:<h1></h1> , <p></p> , <div></div>
b.Empty (Void) Tags:Have only one opening tag (no closing).
Example:<br> , <img> , <hr> , <input>

HTML Elements:

  • Definition: An HTML elements consists of:
  • An opening tag
  • The content
  • A closing tag

Example:

<h1> Hello, World ! </h1>
Here, the element is <h1> Hello, World ! </h1>
  • Nested element: Elements inside other elements
    • <div>
        <p>This is inside a div.</p>
    </div>

    HTML Attributes:

    • Definition: Attributes provide extra information about an element.
    • THey always appear in the opening tag and are written as:
    name = "value"

    Example:

    <img src="image.jpg" alt="Nature image" width="300" height="200"
    • src → specifies image path
    • alt → alternative text
    • width and height → image size

    Example combining all three:

    <p id="intro" class="text" style="color: blue;">
        Welcome to HTML !
    </p>
    • Tag: <p>
    • Element:<p> Welcome to HTML ! </p>
    • Attributes: id, class, style

    Semantic HTML:

    Semantic tags give meaning to your content. Helps in SEO and accessibility.

    <header>Header</header>
    <nav>Menu</nav>
    <section>Content</section>
    <footer>Footer</footer>
