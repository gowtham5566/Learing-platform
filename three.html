Learn Bootstrap Easily 🚀

Build responsive websites faster with Bootstrap

1

What is Bootstrap?

Bootstrap is a popular CSS framework used to build responsive and modern websites quickly.

2

Bootstrap CDN

Add Bootstrap using CDN inside the <head> tag.

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css" rel="stylesheet">
3

Containers

Containers are used to align content properly.

<div class="container">
  Content here
</div>
<div class="container-fluid">
  Full width content
</div>
4

Bootstrap Grid System

Bootstrap uses a 12-column grid system.

<div class="row">
  <div class="col-md-6">Column 1</div>
  <div class="col-md-6">Column 2</div>
</div>
5

Bootstrap Buttons

<button class="btn btn-primary">Primary</button>

<button class="btn btn-danger">Danger</button>
6

Text & Background Colors

class="text-primary"
class="text-danger"
class="bg-dark text-white"
7

Spacing Utilities

Bootstrap provides margin and padding utility classes.

class="mt-5" → margin top
class="p-4" → padding
class="mx-auto" → center align
8

Bootstrap Cards

<div class="card p-3">
  <h3>Card Title</h3>
  <p>Card content</p>
</div>
9

Bootstrap Navbar

<nav class="navbar navbar-dark bg-dark">
  <a class="navbar-brand">My Website</a>
</nav>
10

Bootstrap Forms

<input type="text" class="form-control" placeholder="Enter name">

<button class="btn btn-success">Submit</button>
11

Flex Utilities

class="d-flex"
class="justify-content-center"
class="align-items-center"
12

Responsive Classes

class="col-sm-12"
class="col-md-6"
class="col-lg-4"

These classes help create responsive layouts for different screen sizes 📱

13

Bootstrap Icons

<i class="bi bi-house"></i>

Bootstrap Icons are used to add beautiful icons to websites.
