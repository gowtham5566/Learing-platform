Learn CSS step by step with examples

1

What is CSS?

CSS stands for Cascading Style Sheets. It is used to style HTML elements and make websites look beautiful.

2

CSS Syntax

selector {
  property: value;
}

Example:

h1 {
  color: blue;
  font-size: 40px;
}
3

Types of CSS

1. Inline CSS
<h1 style="color:red;"> Hello </h1>
2. Internal CSS
<style>
h1 {
  color: blue;
}
</style>
3. External CSS
<link rel="stylesheet" href="style.css">
4

CSS Selectors

h1 { color: red; }

.title { color: blue; }

#heading { color: green; }
  • Element Selector → h1
  • Class Selector → .title
  • ID Selector → #heading
5

Colors in CSS

color: red;
color: #7420fb;
color: rgb(255, 0, 0);
6

CSS Box Model

Every HTML element is treated like a box.

div {
  margin: 20px;
  padding: 20px;
  border: 2px solid black;
}
  • Margin → Space outside the element
  • Padding → Space inside the element
  • Border → Surrounds the element
7

Flexbox

Flexbox is used to align elements easily.

.container {
  display: flex;
  justify-content: center;
  align-items: center;
}
8

CSS Grid

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}
9

Hover Effects

button:hover {
  background: #7420fb;
  color: white;
}
10

Media Queries

Used to make websites responsive on mobile devices 📱

@media (max-width:768px) {
  h1 {
    font-size: 25px;
  }
}
11

CSS Animation

@keyframes fade {
  from { opacity: 0; }
  to { opacity: 1; }
}

div {
  animation: fade 2s;
}
